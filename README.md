🛠️ Tecnologías Utilizadas
Este proyecto fue construido utilizando tecnologías front-end modernas y está estructurado con un enfoque "mobile-first".

HTML5: Para la estructura semántica del sitio.

CSS3: Para los estilos personalizados, animaciones (como el botón gradiente) y el diseño.

Bootstrap 5.3: Utilizado como framework principal para el sistema de grillas (Grid system) y componentes responsivos.

Font Awesome: Para la biblioteca de íconos (redes sociales, flechas, etc.).

Google Fonts: Para la tipografía principal del sitio (Montserrat).
